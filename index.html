archive VOD serieses API Documentation

get All Serieses (method GET)

you can find all the serieses that have been ended by entering the following link:

www.your-url.com/getAllSerieses

get Series Data (by ID) (method POST)

yot can find a series by its ID by entering the following link:

www.your-url.com/getSeriesData

parameter: id

get Series By End Date (method PUT)

yot can find a series by its end date (on TV) by entering the following link:

www.your-url.com/getSeriesByEndDate

parameters: day, month
